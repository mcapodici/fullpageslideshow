A full-page background slideshow with crossfade transitions and per-image vertical positioning.

== Description ==

Creates a full-page background slideshow that cycles through selected images using a crossfade effect (z-index swap). Each image can have a custom vertical background position.

Use the `[fullpage_slideshow]` shortcode to display the slideshow on any page or post.

== Installation ==

1. Upload the `fullpage-slideshow` folder to `/wp-content/plugins/`
2. Activate the plugin through the Plugins menu
3. Go to Settings > Full Page Slideshow to add images
4. Add the `[fullpage_slideshow]` shortcode to a page
